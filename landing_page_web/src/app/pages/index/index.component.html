Inovação nascida na FPFTech

Um projeto acadêmico com ambição profissional: propagar saúde através da tecnologia.

O LifeAI é uma solução integrada de bem-estar desenvolvida em Kotlin. Utilizamos Inteligência Artificial para transformar seus dados diários em recomendações personalizadas.

Mobile Nativo

App Android construído com Jetpack Compose e arquitetura MVVM.

Infraestrutura AWS

Orquestração robusta utilizando Kubernetes (K3s) e Docker para deploy contínuo.

Backend & Dados

APIs seguras em Node.js/Django com banco de dados MySQL integrado.
